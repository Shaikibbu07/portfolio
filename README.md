# 🚀 Shaik Ibraheem — Developer Portfolio

A modern, immersive portfolio showcasing my journey as a **Full Stack Developer** and **AI Enthusiast**. The website features cinematic animations, an interactive 3D environment powered by Three.js, and a smooth user experience inspired by premium web experiences.

## ✨ Features

- 🌌 Interactive 3D particle environment built with Three.js
- 🎥 Smooth scroll-driven camera animations
- 🎨 Modern UI with dark/light theme
- ⚡ Command Palette (⌘K / Ctrl+K)
- 🖱️ Custom animated cursor
- 📊 Animated skill bars and counters
- ⌨️ Typewriter text animation
- 📱 Fully responsive design
- 🚀 Optimized for performance
- 🎯 Recruiter-friendly portfolio layout

## 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/Shaikibbu07/portfolio.git
```

Open the project

```bash
cd portfolio
```

Run a local server

```bash
python3 -m http.server 8080
```

Open

```
http://localhost:8080
```
