# 🚀 Shaik Ibrahim — Portfolio

An immersive, cinematic portfolio website inspired by [Active Theory](https://activetheory.net/). Features a scroll-driven 3D particle environment, smooth camera animations, and cinematic section transitions.

![HTML5](https://img.shields.io/badge/HTML5-E34F26?style=flat&logo=html5&logoColor=white)
![CSS3](https://img.shields.io/badge/CSS3-1572B6?style=flat&logo=css3&logoColor=white)
![JavaScript](https://img.shields.io/badge/JavaScript-F7DF1E?style=flat&logo=javascript&logoColor=black)
![Three.js](https://img.shields.io/badge/Three.js-000000?style=flat&logo=three.js&logoColor=white)

## ✨ Features

- **3D Particle Environment** — 2500 particles + wireframe shapes rendered with Three.js
- **Scroll-Driven Camera** — Camera flies through the 3D space as you scroll (lerp-based smooth scroll)
- **Cinematic Section Transitions** — Content fades, scales, and transforms based on scroll progress
- **Ghost Trail Cursor** — Custom cursor with ring + 3 trailing dots
- **Command Palette** — `Cmd+K` / `Ctrl+K` to search & navigate
- **Full-Viewport Projects** — Each project card takes the entire screen with thematic gradients
- **Animated Counters** — Stats count up when scrolled into view
- **Animated Skill Bars** — Progress bars fill on scroll
- **Typewriter Effect** — Cycling taglines with blinking cursor
- **Light/Dark Theme** — Toggle with Three.js fog color switch
- **Konami Code Easter Egg** — ↑↑↓↓←→←→BA triggers Matrix mode
- **Toast Notifications** — For form submit, theme toggle, easter egg
- **Responsive** — Works on mobile and desktop
- **Accessible** — Respects `prefers-reduced-motion`, focus-visible styles

## 🚀 Getting Started

### Option 1: Open directly
```bash
open index.html
```

### Option 2: Local server (recommended)
```bash
python3 -m http.server 8080
# Visit http://localhost:8080
```

### Option 3: Deploy to GitHub Pages
1. Push this repo to GitHub
2. Go to **Settings → Pages**
3. Set source to **main branch**
4. Your site will be live at `https://Shaikibbu07.github.io/portfolio/`

## 🎮 Interactions

| Action | What happens |
|---|---|
| **Scroll** | Camera flies through 3D particle field |
| **Move mouse** | Camera parallax + ghost cursor trail |
| **Cmd+K** | Opens command palette |
| **↑↑↓↓←→←→BA** | Matrix mode easter egg |
| **Click nav** | Smooth scroll to section |
| **Submit form** | Toast notification |
| **Theme toggle** | Switches light/dark + Three.js fog |
